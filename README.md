Diversos scripts simples utilizados no livro "Introdução à Teoria dos Sistemas Composicionais"

*01. SELETOR* - Seleciona aleatoriamente operações que serão aplicadas às alturas de um arquivo musicxml. A seleção, como o próprio nome deixa claro, é uma função que permite ao sistema escolher determinados dados de entrada, em detrimento de outros. A transformação de filtragem funciona como um seletor, por exemplo. O sistema composicional tem em seu setup um seletor, controlado por um processo aleatório, que escolhe entre três operações (+,-,×). O(a) compositor(a) escolhe o fator k, um valor fixo que será incorporado a cada uma das operações (+k,-k,×k). 

02. GSTM - o nome deste sistema é um acrônimo formado pelas iniciais dos nomes das funções: geração, seleção, transformação e mapeamento. Este sistema é tipologicamente original, estático, aberto, com uma entrada de dados e uma saída, ambos discretos. Quatro núcleos básicos integram a arquitetura interna, que correspondem às quatro funções básicas de um sistema composicional. 
Os dados de entrada (E) fornecidos pelo(a) usuário(a) consistem em uma lista de classes de alturas e um fator k. O núcleo seletor (S) recebe essa lista de classes de alturas e seleciona aleatoriamente entre duas transformações (T): Transposição (TK) e Inversão (TKI). Os resultados das transformações passam pelo módulo integrador (Σ_1), para formar uma lista de classes de alturas.
Esses dados de entrada também são enviados simultaneamente a um módulo de mapeamento (M) 11, que os mapeia em valores de colcheia. Isso foi operacionalizado da seguinte forma: foi adicionado o valor 1 a cada um dos valores de entrada, os quais foram posteriormente divididos pela metade, tomando como referência a semínima. 
Um gerador (G) produz tricordes, em uma quantidade determinada pelo ritmo harmônico, o qual, por sua vez, é produzido a partir das durações, as quais são resultado do mapeamento. 
O resultado das operações transformacionais é enviado para um módulo integrador final (Σ_2), o qual também recebe as informações dos módulos mapeador-gerador, produzindo, então, a saída S. Essa saída, em estado bruto, consiste em uma linha de classes de alturas e uma linha de acordes, mostrados em duas claves, que funcionam como um plano para ser utilizado posteriormente em uma composição.
