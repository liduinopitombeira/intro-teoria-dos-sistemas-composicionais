Seleciona aleatoriamente operações que serão aplicadas às alturas de um arquivo musicxml

A seleção, como o próprio nome deixa claro, é uma função que permite ao sistema escolher determinados dados de entrada, em detrimento de outros. 
A transformação de filtragem funciona como um seletor, por exemplo. O sistema composicional tem em seu setup um seletor, controlado por um processo aleatório, 
que escolhe entre três operações (+,-,×). O(a) compositor(a) escolhe o fator k, um valor fixo que será incorporado a cada uma das operações (+k,-k,×k).
